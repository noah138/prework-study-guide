HTML

  • Elements nested inside other elements are called child elements
  • The element that holds all the children is called the parent element
  • Elements included within the head are not visible when a user looks at a webpage
  • The body element contains the conent that is shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Declarations contain two important components: the CSS property we want to apply and the value of the property
  • To apply a rule to all the elements visible on the page, we use a special wild card symbol, or *
  • Two or more elements that will share the same styling can be grouped together with commas
  • When we create an external CSS file, we need to link it to the HTML file by using the HTML element

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • Syncing your GitHub repository with your current work:
    1. git add -A: adds your files
    2. git commit -m "commit message": commits your files
    3. git pull origin main: pulls in the base branch, main
    4. git push origin featurebranch: pushes up your feature branch
    5. In GitHub in your browser, select "Compare & pull request" button. Check that your base branch and comparison branches are correct, enter the name of the PR and a brief description and then select "Create pull request" button
    6. Open the PR you created and merge the comparison branch into the base branch by clicking the "Merge pull request" button.

JavaScript

  • The set of types in Javascript consist of primitive values and objects
  • Common types of variables:
    1. String type - sequence of text
    2. Number type - a number
    3. Boolean type - a true or false variable
    4. Array type - a structure that lets you store multiple values in a single reference.
    5. Object type - everything in JavaScrvipt is an object and can be stored in a variable
  • A good alternative to repeatedly writing the same code is by using functions. You can define a body of code as a function that executes when you call the function name in your code.
  • The console.log method outputs a message to the web console by adding an argument
  • More on arrays:
    1. To create an array, we declare a variable and the assign the variable to a group of values
    2. Arrays are not limited to contain only a single type of data.
    3. The location of an item in an array is identified with a unique number called an index.
    4. The first item of an array has an index of 0
